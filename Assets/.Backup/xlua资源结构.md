### XLua资源结构

- Assets：资源目录
```  
Plugins
    XLua在各个平台运行需要使用到的库文件
XLua
    Doc
        教程文档
    Examples
        示例工程
    Resources
        资源文件夹
    Src
        XLua源码
    Tutorial
        教程示例场景
```
- build：XLua支持库文件的源码
- docs：文档
- General：Tools工具的源码
- Test：测试工程
- Tools：工具
- WebGLPlugins：webGL支持库源码
